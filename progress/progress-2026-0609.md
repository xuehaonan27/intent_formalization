# Weekly Progress — spec-determinism (week of 2026-06-02 → 2026-06-09)

## 1. Last week

**Two-step abstract-determinism pipeline implemented and run (public fns only).**

Step 1 is concrete-completeness (our preceeding work).
Step 2, new this week, generates from each artifact's
`det_check_template` the proof obligation
``s1@ == s2@ ∧ <ensures(s1)> ∧ <ensures(s2)> ⇒ equal(r1, r2)``
and feeds it to Verus with an empty body — the spec must close
the obligation by itself. Only **public fns** are
evaluated — private helpers are scoped out.
