# Dep-graph visualizations

Graphviz renderings of the per-project aggregate dependency graphs
(`../*_deps.json`). Generated by `../../../scripts/visualize_deps.py`.

## Conventions

- **Layout**: top-down (TB). Sources at the top, leaves at the bottom.
  Layer = longest path from this node to any leaf, so each ranked row
  contains nodes at the same "depth" away from the leaves.
- **Color**: classification — `struct` (blue), `enum` (orange),
  `alias` (purple), `union` (cyan), `container` (green; e.g.
  `Vec`/`Map`/`Seq`/`Ghost`), `primitive` (grey), `external` (red).
- **Highlighted nodes** (thick borders) are the `--focus` targets, if
  the view was produced with `--focus`.
- **Red back-edges + dashed red "cycle" box** mark non-trivial SCCs.
- **`<focus>`** views include the focus type's `forward_closure` (what
  it depends on, transitively) and its `reverse_closure` (what depends
  on it, transitively).

| view | size | content |
|---|---|---|
| `nrkernel_full.png` | 45 nodes / 70 edges | full graph; cycle visible in red |
| `nrkernel_focus_Directory.png` | 5 nodes | only `Directory ↔ NodeEntry` cycle + their immediate neighbours |

walker; witness generation must handle it without infinite descent
(e.g. by emitting a fixpoint witness or a bounded-depth view).

## How to regenerate / customize

```bash
# Full project graph
python scripts/visualize_deps.py results-verusage/type_registry/atmosphere_deps.json \
    --out results-verusage/type_registry/viz/atmosphere_full

# Focus on a particular type and its ego-network
python scripts/visualize_deps.py results-verusage/type_registry/atmosphere_deps.json \
    --focus PageAllocator \
    --out results-verusage/type_registry/viz/atmosphere_focus_PageAllocator

# Hide prelude containers / external refs to declutter
python scripts/visualize_deps.py results-verusage/type_registry/atmosphere_deps.json \
    --hide-containers --hide-external \
    --out results-verusage/type_registry/viz/atmosphere_user_only

# Multiple focus types at once
python scripts/visualize_deps.py results-verusage/type_registry/atmosphere_deps.json \
    --focus Page SLLIndex ContainerPtr \
    --out results-verusage/type_registry/viz/atmosphere_focus_PageGroup
```

Each invocation writes three files: `.dot` (raw source), `.svg`
(vector, best for inspection), `.png` (raster, for quick previews).
